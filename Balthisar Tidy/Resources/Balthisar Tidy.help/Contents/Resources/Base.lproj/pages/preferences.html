Preferences

Balthisar Tidy offers several different preferences to control its behavior. The Preferences window consists of multipage pages as reflected below.

Tidy Options
These preferences determine which Tidy options are automatically set when you create a new document or open a document for processing.
File Saving
These preferences can help you protect your original documents in case Tidy does something unexpected by setting the file save behavior.
Miscellaneous
These preferences affect some of Balthisar Tidy’s miscellaneous behaviors.
Updates
These preferences affect Balthisar Tidy’s automatic software updating behaviors, including anonymous system profiling. Note that if you installed Balthisar Tidy From the Mac App Store, then all updates will be delivered from the App Store application, and this Preferences page will not be available to you.
