Choose Tidy options and view descriptions

Tidy offers many options for cleaning your HTML, correcting errors, identifying accessibility issues, and more. All of the available options are available in the document window pane labeled “Tidy Options,” as shown to the left.

Although you can select option values the standard way with the mouse, Balthisar Tidy makes it easy for keyboard users to choose options, too. When the list of options is focused (by either clicking into it our using the tab key), you can move up and down the list with the up- and down-arrow keyboard keys.

You can also use the left- and right-arrow keyboard keys to select option values for most of Balthisar Tidy’s settings. And of course you will see the results in the Tidy HTML view almost immediately!
