Change the Source Area Orientation

Balthisar Tidy for Work has a very configurable display. To toggle the orientation of the Source HTML relative to the Tidy HTML, follow the steps below.

  1. Select Vertical Source Code from the Source Code submenu of the View menu.

  2. You can also easily toggle the orientation using the Orient button on the document window’s toolbar.
