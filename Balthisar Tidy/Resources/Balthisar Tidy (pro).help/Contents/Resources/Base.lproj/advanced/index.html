Use Advanced Features of Balthisar Tidy

Balthisar Tidy for Work offers several advanced features.

Use AppleScript to Control Balthisar Tidy
Learn how AppleScript can be used to automate Balthisar Tidy. If you have dozen, hundreds, or thousands of files to process, then AppleScript can handle the job.
Use Balthisar Tidy as a System Service
Learn how to use the power of Balthisar Tidy in virtually every standard Mac application.
Use Tidy as an Action Extension
Learn how to use the power of Balthisar Tidy in virtually every standard, up to date Mac application.
Use Your Tidy Options with Command Line Tidy
Balthisar Tidy makes it easy to use its settings with the traditional command line Tidy.
Use Themes to Customize your Environment
This feature enables the use of custom colors within Balthisar Tidy for Work, including a full-fledged themes editor.
Export Rich Text Files (RTF)
Exporting Tidy’d text into RTF gives you the ability to easily use Balthisar Tidy for Work’s output in many different environments.
Use a different version of LibTidy
This feature allows Balthisar Tidy to use the Tidy engine from the HTML Tidy project.
