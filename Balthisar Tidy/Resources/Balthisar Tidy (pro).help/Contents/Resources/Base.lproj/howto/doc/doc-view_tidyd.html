View Tidy’d Text

Balthisar Tidy for Work provides the Tidy’d HTML in the Tidy HTML pane. You can view this HTML by following these simple steps.

  1. Make sure that you have opened the document that you would like to Tidy.

  2. Observe the Tidy’d document in the Tidy HTML pane.
