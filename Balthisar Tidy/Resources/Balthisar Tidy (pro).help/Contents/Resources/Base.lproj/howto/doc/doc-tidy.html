Tidy an Existing Document

You can Tidy an existing document with these simple steps, below.

  1. Open the document that you would like to Tidy.

  2. Choose the Tidy options that indicate the cleanup, formatting, and checks that you want Balthisar Tidy for Work to perform.

  3. When you’re satisfied that the Tidy’d text is what you want, save the file.
