Open a Document

You can open an existing document in order to Tidy it by following these simple simple steps, below.

  1. Use the Open… menu item in the File menu.

  2. Drag a file in Finder to Balthisar Tidy’s icon in the Dock.

  3. Create an AppleScript program to tell Finder to open a file with Balthisar Tidy.
