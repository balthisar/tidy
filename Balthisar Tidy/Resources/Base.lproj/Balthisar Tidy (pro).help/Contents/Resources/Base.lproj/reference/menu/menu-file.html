File

Balthisar Tidy for Work uses standard macOS menus that an experienced or novice user should find self-explanatory. The menu items’ descriptions below are limited to the features unique to Balthisar Tidy for Work.

The File menu is a completely standard Macintosh File menu, with the added item Export as RTF….

Note that as is common in many macOS applications, Balthisar Tidy for Work offers an alternate form of the File menu that is displayed when the Option key (⌥) is pressed.

Of special note is the Save Source HTML As… option, which provides you the opportunity to save the contents of the Source HTML panel in the event that you do not wish to accept the Tidy’d version as present in the Tidy HTML panel.
