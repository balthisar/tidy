Create a New Document

By default, Balthisar Tidy automatically creates a new, blank document when it starts, although this behavior can be changed in Preferences.

You can also create a new document by selecting New in the File menu.

New documents’ Tidy options will automatically be set to those in Preferences.

If you are using macOS 10.12 or newer, then documents may be created in a new window, or in a new tab in an existing window. This depends on settings in System Preferences. See Using Tabbed Windows in the Reference section.
