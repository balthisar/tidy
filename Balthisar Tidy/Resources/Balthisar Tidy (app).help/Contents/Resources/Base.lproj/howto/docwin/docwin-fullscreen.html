Use Balthisar Tidy in Full Screen Mode

Full screen mode makes it simple for you to use Balthisar Tidy without any distractions. Follow these steps to enable full screen mode.

  1. Select Enter Full Screen from the View menu.

  2. Alternatively you can use the shortcut key as shown in the View menu.

  3. You can also use the standard Mac OS X document window title bar control to start full screen.
