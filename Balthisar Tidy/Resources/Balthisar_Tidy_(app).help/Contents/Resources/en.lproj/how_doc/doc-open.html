Open a Document

You can open an existing document in order to Tidy it by following these simple simple steps, below.

  1. Use the Open… menu item in the File menu.

  2. Or, Drag a file in Finder to Balthisar Tidy’s icon in the Dock.

If you are using macOS 10.12 or newer, then documents may open in a new window, or in a new tab in an existing window. This depends on settings in System Preferences. See Using Tabbed Windows in the Reference section.
