View and Edit HTML Source Text

  1. Make the HTML Source view active by clicking within it.

    The icons in the left gutter of this editor indicates messages provided by Tidy or the W3C Validator when the Tidy Messages panel or the W3C Validator panel is active, respectively, in the feedback area.

  2. The HTML Source pane is a standard macOS text editor with some advanced features such as syntax highlighting and other functions as available in its contextual menu or in the main menu bar.

  3. You can observe the effects of your edits to the Tidy’d document in the Tidy HTML pane.

  4. You can examine the diagnoses and corrections that Tidy made to the document in the Tidy Messages table.
