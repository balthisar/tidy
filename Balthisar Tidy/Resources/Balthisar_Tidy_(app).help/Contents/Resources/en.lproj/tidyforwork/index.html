Presenting Balthisar Tidy for Work

This is an advertisement. It’s fairly unobtrusive and hidden in a help file, and so we at balthisar.com don’t feel too guilty for plugging a paid product: Balthisar Tidy for Work.

Here’s The Intellectual Incentive

Balthisar Tidy for Work gives you all of the great features of Balthisar Tidy with a few extra goodies to make it worth it:

Customize the Syntax Highlighting and Use Themes
Balthisar Tidy for Work gives you full control over the editor colors and the colors used to syntax highlight your code. Even better you can assign these color combinations to custom themes and change them at will. Several attractive themes are already included in Balthisar Tidy for Work.
Side-by-Side Browser Preview with Optional Scroll Locking
With Balthisar Tidy for Work you can observe the Browser Preview for both the Source HTML and Tidy HTML side-by-side at the same time to ensure that the work Tidy performed did not change how your document renders.
AppleScript Support
Leverage the automation that AppleScript enables and introduce Balthisar Tidy for Work into your complex projects.
Export Tidy’d Output as Rich Text
If you’re working in an environment with code reviews or management reports you might want to have a syntax colored version of your HTML available for use. Balthisar Tidy for Work can output its Tidy’d text into an RTF file.
Export and Import Command Line Configuration Files
Balthisar Tidy has a great UI, but sometimes you need to use command line Tidy as part of your workflow. Why not let Balthisar Tidy for Work help you by using its superb GUI to develop your settings, and exporting them into a ready-to-use configuration for command line Tidy?

And if you already have configuration files for command line Tidy, Balthisar Tidy for Work can import and use them, too.

Top Secret, In-Development Features
We’ve got a lot more planned for Balthisar Tidy for Work. Get your copy from the App Store, now, before the price goes up.

And Here’s the Emotional Appeal

Balthisar Tidy and Balthisar Tidy for Work are fun products to work on, and fun and pride in craftsmanship is really our chief motivation at balthisar.com.

Our primary motivation to charge for Balthisar Tidy For Work is to pay for language localization.

Balthisar Tidy is free and always will be (in fact, it’s free and open source). Sure, some of the development tools cost money and distribution in the App Store requires a yearly, paid membership, but these are relatively low costs that contribute to our enjoyment. But language localization is expensive, much too expensive to eat out of pocket.

So, here’s the deal: 100% of the net proceeds from Balthisar Tidy for Work will be used to pay for language localizations, prioritized by the number of sales in each language region.

And, hey, if you already speak English and don’t care about the Latvians, then consider purchasing Balthisar Tidy for Work on its merits alone.
