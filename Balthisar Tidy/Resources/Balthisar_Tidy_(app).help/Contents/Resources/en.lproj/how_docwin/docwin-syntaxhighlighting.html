Work with Syntax Highlighting

Balthisar Tidy has a very configurable display. To toggle syntax highlighting on and off, follow the steps below.

  1. Select Syntax Highlighting from the Source Code submenu of the View menu.

  2. You can also easily toggle the syntax highlighting using the Highlight button on the document window’s toolbar.

Discussion

What is syntax highlighting?

Syntax highlighting makes it much easier to understand the source code that you are editing as well as the Tidy’d code that Balthisar Tidy produces.

It works by identifying the different parts of the source and Tidy’d documents and coloring them according to their function.

Is syntax highlighting part of HTML Tidy now?

No, it’s a function provided exclusively by Balthisar Tidy to help you work better. HTML Tidy’s emphasis is on correcting and pretty-printing source code for use on servers. Balthisar Tidy makes it easier for you to work on that source code before it reaches the server.

How can I change the colors used for highlighting?

If you want full control over the colors used for syntax highlighting as well as colors for other editor items, please consider purchasing Balthisar Tidy for Work, which gives you control over these items as well as the ability to define and use multiple themes.
