Delete All Balthisar Tidy Preferences

Should you ever want to delete all of Balthisar Tidy for Work’s settings in order to start out fresh, you can delete user defaults and/or delete color scheme files.

Delete User Defaults

You can delete all of Balthisar Tidy for Work’s user defaults quite easily by quitting Balthisar Tidy for Work and then launching it again while holding down the Command-Shift key combination.

Alternatively, simply issue this command in Terminal:

defaults delete com.balthisar.Balthisar-Tidy.pro

Delete Color Schemes

If you use the Colors Preferences to save schemes, then Balthisar Tidy for Work stores data in your home folder:

${home}/Library/Containers/com.balthisar.Balthisar-Tidy.pro/ \
   Data/Library/Application Support/Balthisar Tidy for Work/ \
   Colour Schemes/

You can delete these scheme files manually in Finder, or use the rm command in Terminal:

rm ~/Library/Containers/com.balthisar.Balthisar-Tidy.pro/ \
   Data/Library/Application Support/Balthisar Tidy for Work/ \
   Colour Schemes/*.plist
