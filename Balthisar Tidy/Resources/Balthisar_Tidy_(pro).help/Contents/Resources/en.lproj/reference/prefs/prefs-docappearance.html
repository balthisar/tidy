Document

The Document Preferences panel controls the default appearance and behavior of newly opened document windows.

Preferences Descriptions

Show Tidy options Select this to show Tidy options by default when a document is opened or created.
Show Feedback panel If selected, the Feedback panel containing the Tidy messages table, browser preview, and document validator will be shown upon opening or creating a document.
Show Source HTML and Tidy HTML side-by-side This option lets you control the preferred viewing style for Source HTML and Tidy HTML. When this option is unselected the traditional over-under style will be used.
Don’t show the “wrap” margin in the Tidy HTML view If selected, then the automatic margin indicator that appears when the Tidy “wrap” option is non-zero will be hidden. Uncheck this option to view the right margin indicator.
Automatically create Untitled documents If this option is selected then Balthisar Tidy will automatically create new, Untitled documents when no documents are opened.
