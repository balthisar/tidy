Window

Balthisar Tidy uses standard macOS menus that an experienced or novice user should find self-explanatory. The menu items’ descriptions below are limited to the features unique to Balthisar Tidy.

The Window menu is a completely standard Macintosh Window menu.

This example Window menu reflects the updated appearance since macOS 10.12 (“Sierra”), which enables documents in tabs instead of separate windows. See Using Tabbed Windows in the Reference section.
