Delete All Balthisar Tidy Preferences

Should you ever want to delete all of Balthisar Tidy’s settings in order to start out fresh, you can delete user defaults.

Delete User Defaults

You can delete all of Balthisar Tidy’s user defaults quite easily by quitting Balthisar Tidy and then launching it again while holding down the Command-Shift key combination.

Alternatively, simply issue this command in Terminal:

defaults delete com.balthisar.web.free.balthisar-tidy
