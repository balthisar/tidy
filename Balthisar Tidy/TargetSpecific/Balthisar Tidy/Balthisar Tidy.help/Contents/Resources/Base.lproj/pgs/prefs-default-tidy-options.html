Tidy Options Preferences Page

These preferences determine which Tidy options are automatically set when you create a new document or open a document for processing.

Tidy offers many options for cleaning your HTML, correcting errors, identifying accessibility issues, and more. All of the available options are available in the document window, and you can set defaults for these values in the this Preferences page. These defaults settings will be applied next time you open a document or create a new document.

If you select any of the Tidy options, a description of that option appears at the bottom of the Tidy Options panel.
